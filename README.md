# Proses Web Tech Assignment 

## To Use it follow the instructions

```
$ git clone https://github.com/itsDipaks/EmployeeManagement.git
```

### Installation

```
$ cd Frontend
$ npm install
```

```
$ cd Backend
$ npm install
```

## To Run

```
$ npm run dev 
```
## Features
> - Input Validation for Email and Phone Number
> - Add user Data 
> - Edit userdata
> - Delete User Data 
> - Responsive Ui
> - User Details Table
